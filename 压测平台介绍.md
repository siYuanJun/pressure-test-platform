# 压测平台项目介绍

## 项目概述

本项目是一个功能完善的压测平台，旨在为开发团队提供高性能、可扩展的压力测试解决方案。平台支持多种压测场景，提供直观的测试结果分析和报告生成功能，帮助用户全面评估系统性能。

## 技术栈

### 后端技术
- **Python 3.8+**：使用FastAPI框架构建高性能API服务
- **MySQL**：关系型数据库，存储测试配置和结果数据
- **WRK**：高性能HTTP基准测试工具，用于执行压测任务

### 前端技术
- **React 18+**：现代化前端框架，构建用户界面
- **Next.js**：React框架的全栈解决方案，提供服务端渲染能力
- **TypeScript**：类型安全的JavaScript超集
- **Tailwind CSS**：实用优先的CSS框架

## 核心功能

1. **压测任务管理**
   - 创建、编辑和删除压测任务
   - 支持多种压测参数配置（并发数、持续时间、请求URL等）
   - 任务状态实时监控

2. **测试结果分析**
   - 实时监控压测指标（响应时间、吞吐量、错误率等）
   - 生成可视化图表（响应时间分布、吞吐量趋势等）
   - 支持测试结果导出（PDF格式）

3. **用户权限管理**
   - 多级用户权限（管理员、普通用户）
   - 操作日志记录

4. **系统配置**
   - 压测服务器管理
   - 系统参数配置

## 快速开始

### 环境要求
- Python 3.8+
- Node.js 16+
- MySQL 8.0+

### 安装步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/siYuanJun/pressure-test-platform.git
   cd pressure-test-platform
   ```

2. **后端安装**
   ```bash
   cd backend_admin_python
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

3. **数据库初始化**
   ```bash
   # 创建数据库
   mysql -u root -p < databases_sql/schema.sql
   
   # 初始化数据
   python3 init_db.py
   ```

4. **启动后端服务**
   ```bash
   python3 start_app.py
   ```

5. **前端安装与启动**
   ```bash
   # 管理端
   cd frontend_admin_web
   npm install
   npm run dev
   
   # 客户端
   cd frontend_client_web
   npm install
   npm run dev
   ```

## 使用场景

- **系统性能评估**：全面测试系统在不同负载下的性能表现
- **压力测试**：模拟高并发场景，验证系统稳定性
- **性能瓶颈分析**：通过测试结果定位系统性能瓶颈
- **回归测试**：在系统更新后验证性能是否符合预期
